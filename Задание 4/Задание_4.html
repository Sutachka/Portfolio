Задание 4/Рисование

  • Нарисовать в программе Adobe Animate минимум 3 стилизованных животных, состоящих из простых фигур;
  • Распределить каждый рисунок по отдельным слоям;
  • Анимировать животных с помощью ранее пройденных анимаций движения;
  • Анимация должна представлять собой бесконечный цикл;
  • Добавить фон созданной анимации.

    • Морской мир

    В качестве идеи видеоролика был взять морской мир.

    На экране присутствуют 4 стилизованных морских животных -

    большая черепаха, крабик и две рыбки.

    Черепаха и крабик анимированы с помощью

    анимации движения по кривой.

    Рыбки плывут с помощью классической анимации

    по направляющей.

    Морские обитатели плывут в разные стороны.

    Фон состоит из морского пейзажа с корралами и рыбками.
